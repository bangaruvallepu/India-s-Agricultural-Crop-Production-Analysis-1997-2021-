#  India's Agricultural Crop Production Analysis (1997–2021)

##  Project Overview

This project analyzes India's agricultural crop production data from **1997 to 2021** using **Tableau**. The dashboard provides interactive visualizations to explore crop production, cultivated area, seasonal patterns, and state-wise performance. The project helps users gain meaningful insights for agricultural planning and decision-making.

---

##  Objectives

- Analyze agricultural production trends.
- Compare crop production across different states.
- Study seasonal cultivation patterns.
- Visualize year-wise agricultural growth.
- Build interactive dashboards using Tableau.
